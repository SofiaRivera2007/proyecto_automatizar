¿Qué es HTML? ▶

HTML significa "HyperText Markup Language" y se utiliza para estructurar contenido en la web.

¿Para qué sirve CSS? ▶

CSS (Cascading Style Sheets) se encarga de darle estilo y presentación al contenido HTML.

¿Qué es JavaScript? ▶

JavaScript es un lenguaje de programación que permite agregar interactividad a los sitios web.
